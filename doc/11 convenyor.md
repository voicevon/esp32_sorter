# 传送带系统

## 1. 系统概述
管理传送带上TOTAL_TRAYS个托盘(索引0-(TOTAL_TRAYS-1))，跟踪芦笋直径信息，控制出口分配动作。传送带向右侧移动，托盘依次经过第一至第五个出口区域。

## 2. 核心数据
- **直径和扫描次数数组**：存储TOTAL_TRAYS组数据，每组包含直径值和对应的扫描次数
- **分支点位置数组**：存储5个出口位置
- **出口配置**：5个出口，每个出口对应不同的分类条件

## 3. 工作流程
1. 直径数据插入索引0位置，已有数据依次后移
2. 编码器相位达到85时，检查出口位置索引与托架索引是否匹配
3. 匹配时，根据直径值判断是否触发对应出口动作

## 4. 出口匹配规则
- 出口1：扫描次数 > 1
- 出口2：12mm < 直径 
- 出口3：9mm < 直径 ≤ 12mm
- 出口4：6mm < 直径 ≤ 9mm
- 出口5：直径 ≤ 6mm
