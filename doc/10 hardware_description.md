# ESP32 Sorter 硬件说明

## 1. 硬件组件

- **编码器模块**：100线增量式编码器，用于位置检测
- **直径扫描仪**：单点直径扫描仪，用于物体直径检测
- **出口**：8个舵机，用于控制物体分流
- **上料器**：1个舵机，用于控制物料上料
- **调试模块**：2个按钮和2个LED，用于模式切换和状态指示
- **显示器**：SSD1306 I2C显示器，用于显示系统状态和参数
