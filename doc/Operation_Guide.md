# ESP32 Sorter 操作指南 (更新于伺服剥离后)

本指南说明如何操作分拣系统。**当前系统已彻底移除伺服电机、RS485/Modbus 以及外部电位器调速功能**，全面转向基于电磁铁的出口分拣逻辑。

---

## 1. 系统主模式与导航

系统使用 **编码器旋钮 (HMI Encoder)** 进行主界面导航，并可通过主按钮点击进入。

### 1.1 MODE_NORMAL (生产/运行模式)
用于实时生产分拣。
-   **Dashboard (仪表盘)**：显示当前分拣速率（根/小时）、识别物体数、位移托架计数。
-   **实时直径**：显示扫描仪检测到的最新物体直径（单位：mm）。

### 1.2 硬件诊断子菜单 (Hardware Diag)
进入该菜单后可对硬件进行独立测试：
-   **Conveyor Encoder**：展示传送带编码器的物理脉冲及逻辑位置（0-199）。
-   **Laser Scanner > (激光扫描仪子菜单)**：
    -   *IO Status*：查看 4 路激光的即时电平及跳变计数。
    -   *Encoder Edge*：查看物体经过时的上升沿/下降沿编码器捕获值及脉宽差。
    -   *Raw Diameter*：查看原始脉冲计数值。
-   **Divert Outlet > (分拣出口子菜单)**：
    -   *Single Test*：手动通过旋钮选择并触发特定电磁铁动作。
    -   *Lifetime Test*：电磁铁寿命/老化测试（循环动作）。
    -   *Cycle Drop*：按顺序循环翻转所有出口。
-   **HMI Encoder**：检查旋钮及按键原始逻辑事件。

### 1.3 常规配置 (General Settings)
-   **Diameter Ranges**：通过屏幕配置各出口对应的直径分拣区间，并保存至 EEPROM。

### 1.4 版本信息 (Version Info)
-   显示当前固件编译信息及作者。

---

## 2. HMI 与物理控制

### 2.1 交互逻辑
-   **旋转旋钮**：在菜单中上下移动，或在诊断模式下调整参数（如选择出口）。
-   **按键点击 (Master Button)**：
    -   在菜单层级：进入当前选中的子菜单或动作。
    -   在诊断/运行层级：**返回上一级菜单**。
-   **双击 (暂定)**：部分模式下可用于清除计数。

### 2.2 出口执行器 (Electromagnet Outlets)
-   分拣出口由 **H桥 (L298/TLP250等)** 驱动的电磁铁组成。
-   每秒由状态机控制 `OPEN` 或 `CLOSE` 脉冲（非持续供电，防止烧毁线圈）。

### 2.3 OLED 显示器
-   SSD1306 OLED 提供所有模式反馈。
-   **刷新策略**：采用增量更新与状态缓存，仅在数据变动时刷新，不阻塞中断。

### 2.4 指示灯 (LED)
-   级联位移寄存器 (74HC595) 输出，反映出口当前的逻辑翻转物理位置状态。
