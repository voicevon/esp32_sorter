# 普菲德 (PFDE) PS100 伺服驱动器入门与理解指南

本文档旨在帮助非专业人员快速理解 PS100 伺服驱动器的基本原理、功能逻辑及常见状态。

---

## 1. 什么是伺服驱动器？
伺服驱动器类似于电机的“大脑”。它接收来自控制器的指令（如 ESP32 发出的速度指令），并精确控制伺服电机的转速、位置或转矩。它不仅负责驱动，还会通过编码器反馈实时监控电机的状态（闭环控制）。

## 2. 核心规格与命名 (如何看型号)
以 **P100S-40** 为例：
*   **P100**: 系列名称，适用于 1KW 以下的小功率应用。
*   **S (Simple)**: 简易型，通常体积更小，适配 17 位绝对值编码器。
*   **40**: 功率大小。40 代表 400W 以下，75 代表 750W 左右。

## 3. 三大控制模式 (它能做什么)
驱动器支持三种基本工作方式，通过修改参数 `PA4` 来切换：
1.  **位置模式 (PA4=0)**: 电机像步进电机一样，接收脉冲信号，走到指定位置停止。主要用于精确点位控制。
2.  **速度模式 (PA4=1)**: 类似于传送带控制。你给它一个转速值（如 1000 RPM），它就保持该速度旋转。**（本项目主要使用此模式）**
3.  **转矩模式 (PA4=2)**: 控制电机的“力气”。常用于收卷、张力控制或需要恒定压力的场合。

## 4. 通讯基础 (它是如何听话的)
驱动器通过 **RS485** 总线进行通讯，采用标准的 **Modbus RTU** 协议：
*   **站号 (ID)**: 默认通常为 1。如果总线上有多个驱动器，每个站号必须唯一。
*   **波特率**: 默认为 9600 bps。
*   **使能 (Enable)**: 只有在“使能”状态下，电机才会锁定并响应旋转指令。

## 5. 状态监视 (我们能看到什么)
通过通讯接口，我们可以实时读取电机的各种运行数据：
*   **运行速度**: 电机当前的实际转速。
*   **位置偏差**: 指令要求的位置与实际位置的差值，指示是否有堵转。
*   **母线电压**: 检查输入电源是否稳定（通常为 220V 左右）。
*   **故障代码**: 当驱动器报错（红灯闪烁）时，通过读取代码可以定位具体问题。

## 6. 常见故障自查 (出错了怎么办)
| 现象 | 可能原因 | 建议操作 |
| :--- | :--- | :--- |
| **报警 1 (超速)** | 速度反馈过高或参数设置不当 | 检查负载及增益设置 |
| **报警 2/3 (电压异常)** | 输入电源不稳 | 检查 AC220V 供电 |
| **报警 10 (动力线断开)** | 电机线 U/V/W 没插好 | 检查驱动器到底座的接线 |
| **报警 50 (编码器故障)** | 反馈电缆松动或受干扰 | 重新插拔电机侧反馈线 |
| **电机不转** | 未使能 (Enable) | 检查驱动器面板或指令是否发送了使能信号 |

## 附加：转矩限速 (安全策略)
在**转矩模式 (PA4=2)**下，若电机处于空载状态，转速会持续飞升。
*   **解决方案**: 使用 `PA-31` (转矩模式速度限制，地址 `0x001F`)。
*   **效果**: 无论你设置多大的力度，电机的最高转速都不会超过该设定值（如 600 RPM），从而避免空载飞车，确保设备安全。
