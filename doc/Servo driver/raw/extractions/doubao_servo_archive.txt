# 普菲德 (PFDE) PS100 伺服驱动器 - 核心控制逻辑存档 (来自豆包对话)
存档日期: 2026-03-20

## 1. 核心寄存器定义 (RS485 Modbus RTU)

### 1.1 运行控制类 (实时性高)
- **0x2000 (8192)**: 控制字 (Control Word) - 写
  - Bit 0: Servo ON/OFF (1: 使能, 0: 失能)
  - Bit 1: Alarm Reset (1: 复位报警)
- **0x2001 (8193)**: 状态字 (Status Word) - 读
  - Bit 0: Ready
  - Bit 1: Enabled (正在运行)
  - Bit 2: Alarm Active (发生故障)
- **PA64 (0x0040)**: 目标转矩设定 (Target Torque)
  - 用于转矩模式 (PA4=2) 下的实时给定。

### 1.2 状态监测类 (只读)
- **0x2002**: 实时转矩 (千分比额定转矩, 1000 = 100%)
- **0x2003**: 实时转速 (RPM)
- **0x2006**: 报警代码 (0: 正常)

## 2. 模式切换与初始化逻辑

### 2.1 位置控制模式 (PA4=0)
- 依赖硬件脉冲 (PULSE/DIRECTION)。
- 参数: PA11 (每圈脉冲数), PA14 (脉冲形态), PA15 (转向)。

### 2.2 速度控制模式 (PA4=1)
- 参数: PA22 (速度源), PA40/41 (加减速)。
- 通讯控制需确保来源指向 RS485。

### 2.3 转矩控制模式 (PA4=2)
- 参数: PA32=1 (转矩源设为 RS485)。
- **安全警告**: 必须设置 PA50 (最大转速限制)，防止无负载时发生飞车。

## 3. 推荐操作流程 (状态机实现)
1. **INIT**: 初始化串口, 设置 PA4 为目标模式, 配置 PA50/PA60。
2. **MONITOR**: 持续监控 0x2001 状态字和 0x2006 报警码。
3. **ACTIVATE**: 写入 0x2000 (Bit 0=1) 开启使能。
4. **DRIVE**: 根据当前模式写入目标值 (PA24 为速度, PA64 为转矩)。
5. **SAFETY**: 若转速超过安全阈值或发生报警, 立即清零 0x2000。
